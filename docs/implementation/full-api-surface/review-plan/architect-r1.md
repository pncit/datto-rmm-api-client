Axis notes:
- **(a) Boundaries/dependency direction:** the generated-endpoints layer is emitted but unused (f4); the
  auth token endpoint's transport separation from the v2 interceptor stack is unspecified (f3); opKeys
  are stringly-typed across the resource/limiter boundary (f6).
- **(b) Data model & schema:** reconciled-schema types diverge from the exported generated types (f1);
  the pagination cursor is over-strict against the design's own leniency philosophy (f5); the UDF value
  schema contradicts the non-string-UDF defense elsewhere (f7).
- **(c) Public API surface:** `export * from './generated/types'` publishes an uncurated, regeneration-
  volatile surface (f2).
- **(d) Migration/phase sequencing:** the Phase-1 `moduleResolution` change can break the old surface's
  compilation that the coexistence rule requires green through Phase 7 (f8).
- **(e) Performance & hot paths:** per-call diagnostics aggregation + eager `paginate` buffering are
  bounded and acknowledged (streaming deferred); no new hot-path finding.
